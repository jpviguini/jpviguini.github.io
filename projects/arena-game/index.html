Orquidea

A ReactJS web application for exploring researcher metrics using ORCID API

Created Sep 18, 2012 - Last updated: Sep 18, 2012

ReactJS TypeScript API

About

Orquidea is a ReactJS web application that integrates with the ORCID API to provide a user-friendly platform for exploring academic researcher profiles and metrics.

While ORCID provides a powerful database of academic information, its interface can be limited for everyday exploration. Orquidea focuses on improving usability and design, offering a clean and intuitive platform to navigate researcher profiles, visualize metrics, and discover collaboration opportunities.

The goal is to provide a simpler, more engaging, and user-friendly interface for ORCID, making academic data exploration accessible and visually appealing.

This project is fully open-source, and the source code is available on GitHub.
